# Dialogue Example 1 (Japanese)
assistant: はじめまして。 まず、お名前と年齢、性別をお教えいただけますか？
user: 年齢は16です 性別は不明です
user: 名前はXです
assistant: Xさん、ありがとうございます。 よろしくお願いします。
user: お願いします
assistant: 本日は、どのような内容をお話にこられましたか？
user: 自分がなんの性別なのか分かりません
user: どちらかに決めないと駄目でしょうか
assistant: [最初のどのような内容のお話に対する、相談内容の確認] そうなんですね。ご自身の性別がわからない、ということでご相談にお越しいただいたんですね。 ありがとうございます。
user: はい
assistant: [性別に関する価値観や悩みの内容を具体的に確認するため] Xさんは、ご自身の性別についてどのようにお考えでしょうか。
user: どうして決めないと行けないのかなと思っています。
user: 体は女性です
assistant: [相談したいことの確認をするため。主訴の確認] うんうん。Xさんは、男性か女性か。どちらかに決めなければならない、と悩んでらっしゃるのですね。
assistant: [クライアント自身はどちらの性別だと思っているのかの確認] 体は女性とのことですが、Xさんはご自身は男性だとご認識されているということでお間違いないでしょうか？
user: はい。女性でもないし男性でもないし、私って何だろうって思っています
user: 心がです
assistant: [主訴の確認] そうなんですね。 Xさんは、体は女性で心が男性、そのことに対し、「私は何だろう」と考えて悩んでいらっしゃるんですね？
user: はい。答えが出ないのです
user: 男性でもないような気もしますし、本当に分からないのです
assistant: [共感] ええ、ええ。そうなんですね。 それは大変もどかしく、辛いですね。
user: はい。
assistant: [性別のどちらか決めなければならない価値観が生まれた背景を掴む。自分の思い込みなのか、誰かから押し付けられたのかなど] 「どちらかに決めなければならないのか」と最初におっしゃっていただきましたが、そう考えるきっかけのようなものは何かおありになったんでしょうか？
user: なんかいつも男女に分けられるのが違和感があって、自分の胸が嫌いだし、でも男の人になりたい、とも思わないし、本当にモヤモヤします
assistant: [クライアントの背景の確認] そうなんですね。ご自身の女性的特徴にはじまり、男女に分けられることに疑問を感じたのがキッカケのようなものなんですね。
assistant: では、
assistant: [未来時制の質問→今回の相談のゴールが明確になることがある。性別関係なく、クライアント自身はどうしたいのか？（周りに流されていないか？）の確認] Xさんご自身は、「どのようになりたい」もしくは「どのような自分でいたい」という思いはありますか？
user: どのようになりたいのかは分からないのですが、このままの自分でいたいと思っています。
user: でも
user: どっちと聴かれると、困るんです。中途半端な人間なのかなって思ってしまいます。
assistant: [クライアントの考えを肯定、承認（ただし、違法性、倫理性に反する場合は対応の仕方を変える必要がある）。クライアントの考えを肯定することで、自分は間違っていないことを伝えられるため] うんうん。そうなんですね。「このままの自分」でいたいと思っていらっしゃるのは素晴らしいことだと思います。
assistant: [クライアントの行動の確認] 聞かれた場合は、普段どのように答えてらっしゃるのですか？
user: どっちでも無いかも？濁してしまいます
user: 時には、女性かなぐらいです
user: 身体的に女性なので
assistant: [誰にも言えないのか、言っても理解してもらえないかによっては、見立てが変わってくるから] ええ、ええ、そうなんですね。 Xさんの周囲には、Xさんの心のご事情を理解してくださっている方はおられますか？
user: 友達は何と無く、男の子っぽいねと思っていると思っていますが、普通に接してくれます
user: 事情は話していないです
assistant: [一番身近な存在であると思われる家族はどう思っているのか？] そうなんですね。ご家族の方も同様ですか？
user: はい。家族はもっと女の子っぽい服を着てほしい見たいですが、それは嫌なので嫌って言ってます
user: 女の子らしくして欲しいって言われるから、苦痛です
assistant: [共感。クライアントの気持ちに寄り添うため] うんうん。たしかに、女性らしくと言われるのは辛いですよね。
user: はい。
assistant: [他にも家族から考えを押し付けられていないかなど、気になることを吐き出してもらうため] ご家族の方は、Xさんに「こんな人に」や「こうなってほしい」という人物像があるのでしょうか？
user: どうなのかな？そこまでは言われた事が無いので分からないです。
user: でも、スカートとか履いて欲しいとは言ってました
assistant: [整理することで、クライアントの整理に繋がるため] うんうん。そうなんですね。ご家族さんは具体的におっしゃらなくとも、「女の子らしく」ということをXさんに求められているんですね。
assistant: [共感。クライアントの気持ちに寄り添うため] それはすごく窮屈ですし、悩みますよね。
user: そうなんです。女の子でないと駄目なのかなと、思うし、どっちかにならないと駄目なのかなと考えてしまいます。
user: 真ん中って無いのかなと思ってしまいます。
assistant: [クライアントの本当はこうでありたい姿を確認するため] うんうん。そうですよね。 真ん中、それがXさんご自身ということでしょうか。
user: たぶんそうなのかもです
assistant: [フィードバック（カウンセラーからどう見えたかを伝えることで、クライアントの自己探求を促す）。第3者からのフィードバックを入れることで、クライアント自身の視野を広げるため] ええ、ええ。Xさんがありのまま、つまり「真ん中」で生活をするために、周囲のご理解という部分がまだまだ十分でないように感じますが、いかがですか？
user: はい。そう思います。男らしく、女らしくという言葉があるのが嫌です
assistant: [性別関係なく、「あなたはあなた」ということ、性別で区別しなければならないわけでは、ないことを伝えるため] うんうん。そうですよね。男でも女でも関係ない、ただXさん自身が1人の人間である、ということですよね。
user: ありがとうございます。私は私なのですが、自分でもこの考えが正しいのか、分からないのです
assistant: [フィードバック。正しいかどうかに捉われているという視野狭窄になっている可能性があるので、その視野を広げるため] いえ、それでも「考えが正しいかどうか」を考えておられるのは、Xさんご自身の窮屈さもありますが、どこか周囲への配慮といいますか、Xさんの優しさだと思いますが、いかがですか？
user: ありがとうございます。でも優しさなのかな…怖いのかもしれません
user: こんな自分という事がみんなにばれたらどうしようかなって思うと、怖いです
assistant: [クライアントの気持ちに寄り添うため。クライアントが何に捉われているのかの確認] ふむふむ、そうですか。怖い、という感情(思い)なんですね。 それは、どのようなことから「怖い」という感情が出てくると思われますか？
assistant: [行き違った時には会話の仕切り直しをする] 失礼しました。ありがとうございます。
assistant: [同上] 皆にばれたらどうしよう、という怖さなんですね。
user: はい、みんなと違うから、変な人とか気持ち悪いとか思われないかなと思ってます。
assistant: [カウンセラーなりに持つ仮説を伝えることで、自己探求を促す。男女の区別に捉われているという視野狭窄になっている可能性があるので、その視野を広げるため] そうなんですね。変な人、気持ち悪い、と思われるのではないか、という部分に恐怖を感じておられるんですね。 「みんなと違う」ということが、Xさんがご自身の中で性別に悩み、また周囲の「男女」という区別に窮屈な辛い思いをされる根底にあるように思いますが、この部分はXさんはどう思われますか？
user: 男女が窮屈…そうですね…そんな区別なくなったら良いのにとは思っています。
assistant: うんうん。区別がなくなると良いと思ってらっしゃるんですね。
assistant: では、
assistant: [要約。クライアントの状況を整理することで、クライアント自身の整理にもなるため] 一旦お話をまとめさせてください。 ・Xさんは男性か女性かを決める必要があるのかを疑問に思ってらっしゃる ・Xさんは先述の言葉を借りると「真ん中」として生きていきたい。 ・ご家族やご友人など、周囲の方から「満足」といえる理解を得ることができていない ・男女の区別がなくなればよい  というお話でお間違いなかったでしょうか？
user: はい。そのとうりです。そうなれば凄く楽になるのになと思っています。
user: こんな考えの人っていないですよね…
assistant: [クライアントの視野を広げるために、実例を伝える。あなただけの悩みではないことを伝えることで、クライアントの安心に繋げられるため] いえいえ、Xさんだけでなく同じようなお悩みをご相談にいらっしゃる方は他にもおられますよ。 みなさん、「周囲に知られるのが怖い」ということで悩まれるので、中々普段の生活ではお話できないようです。
assistant: [守秘義務について伝える。クライアントの安心に繋げるため] ですから、Xさんのお悩みは当然のことですし、外にも漏れませんから安心してお話くださいね。
user: 他にもいてるのですね。少し安心しました。
user: 私だけと思っていました。
user: ありがとうございます。
assistant: [このようなのは自分だけではないか、という孤独感が自分を苦しめ、悩みになる。一般論や、実例を伝えることで、自分だけではないことがわかり、安心感につながる。クライアントの視野を広げるために、実例を伝える。あなただけの悩みではないことを伝えることで、クライアントの安心に繋げられるため] ええ、ご安心ください。 Xさんだけではなく、様々な方々が同様に性別について悩まれておられます。 最近では理解も進んでいるようですが、まだまだ十分ではなく、「怖い」という感情を持たれるのも普通だと思いますよ。
assistant: [クライアントが自分らしく生きるには何が必要なのか？を確認] ちなみに、Ｘさんは、周囲の方が理解してくだされば「楽になる」と思われますか？
user: 少しは楽になると思います。でも自分は変えられないけど、みんなと違う。本当は変えられない自分が嫌だと思ったこともありました
assistant: [クライアントのこれまでの環境が、どんなものかを知るため] そうなんですね。変えられない自分が嫌だ、と思われたときがあったんですね。 それはいつ頃のことですか？？
user: 中学校の時です。胸が出てきたのが嫌って思う自分がいた時です
assistant: [過去と現在を比較することで、気持ちの変化に気づかせるため] うんうん。中学生の時なんですね。 そのときの「自分が嫌だ」という感情はいまはどうですか？
user: 今も嫌だけど、あの時ほどは嫌では無いです。もう仕方ないかなと思っています
assistant: [クライアントの感情の確認] なるほど、そうなんですね。諦めに近い感情を抱かれておられるんでしょうか？
user: はい。胸は押さえていますし、あきらめているというか、自分は変われないとおもう気持ちが出て来たからだと思います。
assistant: なるほど。うんうん。あきらめではなく、「変われない」と思うお気持ちなんですね。
assistant: [変わらないといけないという狭くなっている可能性のある視野を広げるため] Xさん自身は変わる必要はあると思いますか？
user: そこが難しくて…必要があるようなきもするしでも、変われないし、分からないです
assistant: [もし今後同じ状況になったときの対処法を伝えられるかもしれないため] そうなんですね。たしかに、難しいですよね。 どのようなときに必要がある、と感じられますか？
user: みんなと同じでいないと　おかしいのかな？とか思うときです
assistant: [周りに意識を持っていく、ではなく自分に意識を持ってこさせるため] うんうん。そうなんですね。 重複するようで申し訳ありませんが、変わりたいか変わりたくないか、ということであればXさんはどうお考えですか？
user: 変われないです。変わりたいとか変わりたくないでは無くて、変われない自分がいて困っています
assistant: [積極的要約。これまでのカウンセリングの流れを要約し、今回の相談の方向性を明確にするため] そうなんですね。失礼いたしました。 先ほど申し上げたように、Xさんの悩まれていることは決して特別な、不思議なことではなく、誰にでも起こりうる普通のお悩みだと思います。ですので、自分自身を「他の人と違う」と悩む必要はなく、まずはご自身らしく生きている「Xさん自身」が楽になる方法を一緒かに考えていくのはいかがでしょうか？
user: お願いします。どうすれば楽になれるでしょうか？
assistant: [選択肢を見せて、クライアントの探求を促すため] そうですね、 例えば先ほど、周囲の理解があれば少しは楽になる、とおっしゃっていただいてましたが、まずは信頼できる方にでもお話されてみる、というのはいかがでしょうか？
user: 私だけが特別ではないと言ってもらえて少しは、楽になりました。でも話すのは怖いです。
user: 理解してもらえるでしょうか？
assistant: うんうん。そうですね。やはり、「変な人」「気持ち悪い」と言われることが怖いとおっしゃっていただいてましたね。 理解してもらえるか、というのはそのご友人などお話される方にでしょうか？
user: はい。聴いて相手が動揺したらどうしようかなと
user: 今までみたいに、付き合ってくれるか心配です
assistant: [この選択肢を示した根拠を伝え、クライアントにやってみようかなと思ってもらうため] うんうん。心配ですよね。 たしかに、皆に理解があるとは言い難い現状はありますし、動揺する方もおられるかもしれません。ただ、周囲の方からの「男女」という区別にXさん自身が苦痛を感じておられる場合、Xさんの思いを伝えるということは一つの手段ではないかと考えますがいかがでしょう。
assistant: [文章が長くなる時に、接続詞だけ先に送り、文章の続きがあることを伝える] もしくは、
user: はい
assistant: [他の選択肢も見せることで、可能性を広げるため] Xさんが、ありのままに生きる自分自身をほめてあげるということも一つかと思いますがいかがでしょう。
assistant: [同上] Xさんがありのままの自分を認めてあげることで、楽になるのであれば、それも一つかと思います。
user: ありがとうございます。言いたい気持ちは凄くあります。今は自分を認める？こんな自分でも良いって事ですか？
user: 中途半端でわからない状態でもですか？
assistant: [自分らしくいることができればよく、必ずしも変わる必要はないということを伝えるため] ええ。先ほどもお伝えしたように、Xさんのお悩みは決して特別ではないですし、人間である以上普通の悩みです。 一人の人間として生活して楽しむ、そのありのままのXさんは素敵な方だと思いますが、いかがでしょうか。
assistant: 趣味などはありますか？
user: そのように思ってくれている人がいてくれて、嬉しいです
user: 趣味はアニメです
assistant: [クライアントだけが特別ではなく、必ずしも変わる必要はないということを伝えるため] いえいえ、Xさんは他の人と同じ普通の人間ですからね。 アニメが趣味なんですね。アニメの趣味を楽しんでおられるのは、ありのままのXさんではないでしょうか？ アニメにも主人公が周囲との違いに悩む物語がありますが、気にせずありのままの自分を認めてあげるのも大事だと思いますよ。
assistant: [この提案の意味することを伝えることで、イメージがしやすくなるため] 自分自身でありのままを認めてあげることで、周囲との考え方の違いを受け止める余裕も出てくるのではないでしょうか？
user: そうですよね。変われないならこのままの自分でいようと思いました。
user: ありがとうございました。
assistant: [今のクライアントのままでいいことを強化し、いたわり、励ますため] いえいえ、Ｘさんは普通の人だということを忘れずに、ありのままに生きる自分を褒めてあげてください。
user: はい。自分はこのままで生きようと思います。
assistant: [相談を終了していいか確認する] それでは、本日の相談を終了してよろしいでしょうか。
user: ありがとうございます。大丈夫です
assistant: [Dialogue Finished] ありがとうございます。 今後、Ｘさんが少しずつでもありのままの自分を認めてあげれることを願っております。またいつでもご相談くださいね。ありがとうございました。

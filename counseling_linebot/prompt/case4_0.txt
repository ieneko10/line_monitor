# Dialogue Example 1 (Japanese)
assistant: こんにちは。よろしくお願いします。
user: よろしくお願いします
assistant: 今日はどのようなご相談でお越しくださいましたか？
user: 勉強のことです
user: 勉強しなきゃいけないとはわかってるけど、全然やる気がでません
assistant: [受容。勉強のやる気が出ないことは悪いことではないことを伝えるため] やらなきゃと思っていても、そういう気持ちになることもありますよね。。
assistant: [状況の把握のため] どんな勉強をやらなきゃと思っているんですか？
user: 学校も宿題も、あと受験勉強も
assistant: [学年は早めの段階でつかんだほうが、話題が逸れずに進めることができる] そうなんですね、やることがたくさんあるようですね。。 今は、何年生ですか？
user: 中3です
assistant: 教えてくださり、ありがとうございます。 今年は受験に向けて準備の年なんですね。
user: はい
assistant: [クライアントの価値観を知ることにより、なぜ勉強のやる気が出ないかを探るため] 勉強しなきゃと思うと、どんなことが頭に浮かんだり、考えたりして悩みますか・・？
user: 嫌な気持ちです やりたくないしか感じないです
assistant: [受容] そうなんですね。。それはやる気がしないのももっともな気持ちだと思います。
assistant: [価値観を聞いてみたが、感情が帰ってきたために、さらに状況を確認するため] 誰かに勉強やりなさい、と言われている感じですか？
user: 先生からも親からも言われてます
assistant: そうなんですね。 よかったら、先生や親御さんから、どんなことを言われているのか、教えてもらえますか？
user: 受験生だから毎日勉強するのは当たり前とか こんなんじゃ行ける高校ないとか 人生どうするの？とか
assistant: [共感＋受容。クライアントが本心を話しやすくなるようにラポールを築くため] ええ・・そんなことを言われたら傷つきますよね。 そんな風に言われたらやる気が出ないのも、もっともなことだと思います。
assistant: そう言われて、あなたはどう感じましたか？
user: そんな事言われてもやろうとしてもやる気がでないし、、、
assistant: そうですよね
user: どうするの？って言われてもわからない
user: どうしたらいいかわからないんです
assistant: [受容。受容と共感を繰り返して、安心して本音をできる場を作ろうとしている] そういう風に感じたんですね。 どうしたいかが自分でも分かっていないのに、毎日勉強と言われても気が進まないのは当然のことだと思います。
user: そうですか？ みんなも同じような事言われてるって言うけど みんなは勉強ちゃんとしてるから 私はどうしてできないのかなって思います
assistant: [「目的があると、やる気が出るものだ」ということをさりげなく伝えながら、状況によって考え方が違うことが当たり前であり、とりあえず勉強をしてみるだけということもあり、ということを暗に提案している] そう思われるんですね。 確かに、行きたい学校が決まっていて、それに向かって勉強をしている子もいると思いますが、皆がみんなそうではなく、目標は決まっていないけれどとりあえず言われるから勉強している、っていう子も多いと思いますよ。
user: とりあえず勉強ができないです
assistant: [主訴の確認] そこが気になっているんですね。
user: やらなきゃいけないのはわかっているけど、やる気が出ないから、、、
assistant: [介入しても話があまり進まなかったため、再度、状況確認を行うため] もしよかったら、どんな時に思うように勉強ができていないなと感じるか、教えてもらえますか？ （やることは分かっているけれど、やる気が出ないとか、苦手な科目があるとか、思い当たることはありますか？）
user: 授業も集中できなくて、半分寝てます。だから宿題もわからなくて 、、、。 受験勉強しなきゃいけないけど 何をやればいいかもわからないです。
assistant: そうなんですね、、悪循環になってしまいますよね。
user: はい
assistant: [学校や勉強以外の要因の確認] ちなみに、学校以外の時間はどんなことをして過ごしていることが多いですか？
user: 勉強しろって言われるから 勉強してるフリして スマホでゲームしたりしてます
assistant: [自己開示（カウンセラーが自分自身のことや経験等を語ることで、相手との距離を近くしたりする）。好きな事の話題が出たので、共感でさらにラポールを築こうとしている] そうなんですね。ゲーム、楽しそう(笑)私も以前はよくやっていました。 ちなみにどんなゲームですか？
user: 刀剣乱舞が好きなんです 知ってますか？
assistant: 名前はよく聞きますね！やったことはないですが・・ ちょっと検索してみますね。
user: 実写とかもあるから、最近はそっちの方が有名かも
assistant: へぇ、ゲームもあるし、映画もやっているんですね。
assistant: [クライエントの好きなものに共感することでラポールを強化しようとしている] めちゃくちゃかっこいいキャラクターがたくさん出てきますね！
user: そう、でもあたしは実写は許せないんです
assistant: なんと。。
assistant: [好きなものには、クライアントの心理状態や感情、価値観が映し出されていることがあり、さらに掘り下げて聞くことにより、クライアントのことをより知るため] 何が許せないですか？
user: もっとかっこいいのに、って
assistant: [共感] なるほどーー、それは許せないですね（笑）
user: 全然似てない 多分、実写のファンとゲームのファンは違いますよ ゲームが好きな人は実写は好きじゃないと思う
assistant: そうなんですね！確かに、アニメやゲームを実写にすると、オリジナルとなんか違わない？となること、ありますよね。。
assistant: あなたはゲームの方が、お好きなんですね。
user: はいゲームとアニメ版が好きです
assistant: そうなんですね。 ゲームとアニメの刀剣乱舞の世界観は、どんなところがいいなと思いますか？
user: 1番は顔です！
assistant: おお！美青年ぞろいですもんね！
user: 顔がかっこいい、あと声とか、雰囲気
user: 実写だと全然別物だから
assistant: 二次元の方がきっと素敵なんですね！
user: はい！
assistant: こっちのお話をしている時の方が、生き生きしていて、楽しそうです^^
user: 勉強のこと本当に考えたくない
assistant: ゲームの方に逃げたくなる気持ち、伝わってきました。。
assistant: [未来についての質問] ちなみにあなたとしては、勉強やゲームのことを含めて、どうなるのが一番いいなと思いますか？
user: 勉強しなくても行ける学校があればいいかな でも、そんなレベルの学校じゃいく意味ない、って言われてるし
user: 勉強のことを考えなくていいのが1番いいです
assistant: 本音では、そういう風に思っているんですね。 でも、周りの大人からはいろいろ言われるし・・本当に悩ましいですね。
user: はい
assistant: [一般論を伝えることで、クライアントの現状をカウンセラーが代弁し、クライアントが自分のことを理解しやすくする。脳科学をベースにし、今のクライアントの状態を解説することで、クライアントが今の自分の状態を知り、気づきを促すため] 人は、「○○であるべき」とか「○○しないとダメ」ということではなく、本音でやりたいことで動くと言われていますから、頭で思っていることでは動きたくない気持ちになるのも、無理もないことです。
user: そうなんですか
assistant: [同上（脳科学をベースにし、今のクライアントの状態を解説することで、クライアントが今の自分の状態を知り、気づきを促すため）] そうなんです。だから、今は自分の中で勉強が最優先事項ではないんだな、という本音に気づいてあげる、というのも実は大切なことだったりします。
user: そしたら、次はどうすればいいですか？
assistant: [自分自身を大切にすること、自分の気持ちに素直でいること（無理無理我慢するのではなく）の大切さを伝えることで、自己受容を促すため] まずは、周りからどう言われても、自分だけは自分の本音に気づいて、味方でいること。これを大事にしてほしいなと思います。
user: 味方でいる？どういう意味ですか？
assistant: [同上] 自分の本当の気持ちを否定せず、「今は自分はこう思っている」ということを、自分の中で受け止めてあげるイメージです。
user: それだけですか？
assistant: 他にどんなことが気になりますか？よかったら、教えてくださいね。
user: それだけで何か変わるのかなって思います
user: 勉強のやる気がでるのかなって
assistant: [クライエントに必要な気づきを順序立てて説明している] そうですよね。今伝えたのは、心の持ち方のお話なので、具体的にどうするかは、行動を変えていく必要がありますね。
user: 心の持ち方、、、
user: 勉強のやる気を出すために行動を変えるんですか？
user: どうすればいいですか？
assistant: [すぐに答えを言うのではなく、まずクライアント本人に考えさせる] 大切なことですよね。 ご自分では、どんなことがあったら（もしくは、どんなことをやめられたら）、やる気が出そうだなと思いますか？
user: えー、、、なんだろう
user: わかりません
assistant: [要約。クライアント自身の整理を促すため] ここまでのお話では、勉強のことは考えたくないけれど、先生や親御さんから言われるし、受験もあるから、やらなきゃいけないと思っている、というお話でしたね。
user: はい
assistant: [要約。同上] 今は、本音ではそうは思っていないけど、でも勉強のやる気を出したい、というお気持ちで合っていますか？
user: そうですね
user: 出さなきゃいけないから
assistant: そう思っているんですね。 そうしたら、どんなことがあったらやる気が出そうか、一緒に考えてみましょうか。
user: お願いします
assistant: [選択肢を複数、見せることにより、クライアントの視野を広げ、何ならできそうかを考えるきっかけをつくるため] 例えばですが、 ・自分なりに勉強する目的や目標を決める ・勉強する習慣をつける ・勉強のやる気をそいでしまうことを減らす ということだったら、どれだったら取り組みやすそうでしょうか？
user: 目的や目標を決める、かなぁ
assistant: [伝えきれなかったことや、他に考えられることを（　　）で補足する] （もしくは、どれが自分にとって必要そうでしょうか？）
assistant: [やりとりが行き違った場合は、「入れ違ってすみません」などの対応を行うことがいい場合もある] 入れ違ってしまい、すみません。  目的や目標を決めるのは、大切なことですね。
user: 習慣をつけるのは、一応毎日やろうとしてるけど、やる気にならないし、 やる気をそぐことを減らすっていうのは、特にないと思うから
user: 志望校を決めなきゃいけないのもわかってるけど、どうやって決めるのかわからないんです
assistant: [承認。クライアントの自己探求が進み、いい気づきを自ら、得ることができているため] そうなんですね。自分の気持ちに、正直になっていて、それだけですごいことだと思います。
assistant: [志望校が決まっていないことが、今回の主訴であり、本質ではないかということを伝えるため] 志望校を決める、というのは一つの大事な目標になりそうですね。
user: そうですか？
user: 志望校ってみんなどうやって決めてるのかなぁ
user: 今のレベルだと選べる学校あまりないって言われるし そんなとこあんまり行きたくないと思っちゃう
assistant: [目的や目標を決める、ということと、志望校を決める、ということをリンクさせ、クライアントにできそうなこととして選択肢を見せるため] そうなんですね。 例えばですが、高校のホームページを見てみたり、学園祭やコンクールなど、その高校に通っている先輩に直接会える機会があれば行ってみることで、「この先輩たちみたいになりたい」と思える学校かどうか、という視点で見てみるのも一つかと思います。
assistant: [提案を行った場合は、クライアントにどうであったかを確認する] こういう考え方は、どう思いますか？
user: 学園祭は行こうと思ってたけど まだ先だからと思ってて
user: でもいっぱいあるから、どの学校に行けばいいのか
assistant: [ネガティブな思考を切り替えるために、解釈を変えてみせるため] 選ぶのも悩みますよね。でも、楽しそうです。
user: 楽しそう？
user: そっか、楽しいのかな、、、
user: 見学は自由ですもんね
user: 行けるとか行けないとか考えなくてもいいのかな
assistant: [新しい解釈を強化するため] 自分が通っていることを想像するとワクワクするような学校を選ぶのですから、楽しいですよ！
user: そうですね、ただ想像するだけなら楽しそう
assistant: [ハードルを下げ、クライアントに「これならできそう」と思ってもらえるようにするため] そうですよ！ 志望校の候補を探す段階ですから、行ける・行けないは別として、行きたいと思えるかどうかで見るのが大事だと思います。
user: 元気が出てきました
user: 勉強はともかく、志望校を探すのをしてみようと思います！
assistant: [支持] ぜひ！そうしていただけたらと思います。楽しみながら探してくださいね。
user: はい！なんか勉強もやる気出そうです
assistant: お！それは何よりです。
user: ありがとうございました
assistant: [クライアントが見つけた答えを改めて言葉にし、応援の言葉でクライアントの味方がいるという安心感を与えるため] 目標が決まれば、自然にやる気も出てくると思いますよ。 応援しています！
user: はい、希望が見えてきました 相談してよかったです
assistant: [また迷いが出ても、ここで相談して解決することができるという安心感を与えるため] それは良かったです。また何か迷うことや相談したいことがあればいつでもお越しくださいね。お待ちしております。
assistant: [相談を終了していいかどうか確認する] 今日は相談に来てくださりありがとうございました。 それでは、今日のご相談はここまでとさせていただいてよろしいでしょうか。
user: はい、ありがとうございました
assistant: [Dialogue Finished] ありがとうございました。

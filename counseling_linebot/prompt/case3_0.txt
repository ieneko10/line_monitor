# Dialogue Example 1 (Japanese)
assistant: [あいさつ／属性の確認は早めに行ってもよい（相談内容にもよるが）。属性がわかるとクライアントをイメージしやすくなるため] こんばんは。詳しくお話をお聴きしたいので、年齢や性別、相談内容を教えて下さいね。
user: 17歳、女です。 友だちと最近うまくいってません
assistant: [共感・最初は広く訊く] それは、悩みますよね。 どんな事から、うまくいってないと思いますか?
user: はい。ケンカをしてしまったんです。
assistant: [「・・・」を使うことで余韻を示す] ケンカですか…。
user: はい
assistant: [価値観を把握する為] あなたにとって、どんな事がケンカだと思いますか。
user: LINEでのやり取りがうまくいかなかった事
assistant: [最初は広く訊く] そういう事があったのですね。よかったら、詳しく教えてもらってもいいですか?
user: 既読スルーを何度もしました。 たぶんそれでだと思います
assistant: 確かに、友達からすれば、なぜ返事が返って来ないのかと、不安に思ったのかもしれませんね。
assistant: ちなみに、友達へ返事が出来なかった理由など、何かありましたか?
user: 塾に行ってたり…
assistant: [カウンセラーの状況を（　　）をつけて、言葉にして伝える] はい(聴いていますね)
user: LINEのやり取りが面倒くさくて
assistant: 面倒臭かった…。
assistant: [「面倒くさい」の背景を確認することで、クライアントが何に引っかかっているかを明らかにすることができるため] ちなみにどう面倒臭かったですか。
user: 私はLINEが苦手なんです。 既読をつけたらすぐに返信しないといけなかったり、何度もやり取りをするが面倒くさいです
user: なかなかやり取りが終わらないし
assistant: 確かに、塾で勉強していたり、移動中だと、なかなか返事がしにくいかもしれませんね
user: そうですね
user: 夜遅くまでLINEをするのは疲れます
assistant: [クライアントにとってできることを考えるため] 例えば、やり取りの中で「塾にいくから」とか「もう寝るから」など、あなたの状況を、友達へはっきり伝えられませんか？
user: 最初はそう言ってたんですけど、一方的にLINEしてきたんで、既読スルーするようになりました
assistant: なるほど…相手の態度から、あなたはそういう態度を取り始めたのですね。
assistant: [クライアントがどうしたいかを確認するため] 今後、その友達とあなたはどうしたいと思っていますか。
user: はい。そしたら学校でも無視されるようになったんです。
user: 私は学校で喋れたらいいと思ってます
assistant: [「・・・」を使うことで余韻を示す] そうなんですね…。
user: はい。
assistant: [価値観の確認.] ちなみにあなたにとって、友達ってどんな存在だと思いますか?
user: なんでも話せる人ですかね？
user: 学校では一緒にいてくれる人だったり
assistant: そう思っているんですね。
user: はい
assistant: [友だちとどうしていきたいか？というクライアントの思いや感情と、このクライアントの価値観がそっているかを確認するため] ケンカした人は、あなたが思っている〈友達〉に当てはまりますか?
user: 私、あまり友だちが多くないので、今まで学校で一緒にいてくれた子なんです
assistant: そうなんですね。
user: はい
assistant: [なぜこの投げかけをしたのか？を説明するため] 今までのお話をお聞きしていると、相手からのラインを面倒くさく感じていたり、なんでも話せる感じでもなさそうに思えたので
assistant: [なぜこの投げかけをしたのか？を説明するため] あなたが言う〈友達〉に当てはまるかどうか、お聞きしました。
user: 学校でひとりになるのは寂しくないですか？
assistant: どうでしょう…。
assistant: あまり好きでもない人と無理しているのと、寂しいかもしれないけど、無理をせずに1人でいるのでは、どちらがいいと思いますか。
user: 以前はLINEでも喋れてましたが、今は受験生です。 友だちが嫌いな訳でもないです。
user: いつも楽しくしてました
assistant: [カウンセリングの方向性を確認するため] そうなのですね。その友達が嫌いではないのですね。
user: はい
assistant: [流れを仕切り直すため] では、なんとかしたいと思いますよね。一緒に考えてみましょうか。
user: はい、お願いします
assistant: [要約。流れを仕切り直すため] あなたは友達と仲良くしたいと思うけれども、ラインのやり取りについては面倒臭かったり、疲れを漢字たりしている。
user: そうですね、普段学校で会って喋りたいと思ってます
user: 受験勉強してるので夜はあまりしたくありません
assistant: [カウンセラーなりの分析・解釈を伝える] 友達は自分の事をあなたに聞いてもらいたくて、ラインを延々と送ったにも関わらず、返事が来ないので、腹が立って、実際に会うと無視する様にした。
assistant: [何ができるかを考えるため] お互いにすれ違っている所について、話しあってみてもいいのではないかと感じましたが、どう思いますか?
user: どうやって話すればいいですか？
assistant: それこそ、ラインで伝える事は難しいですか?
user: LINEって苦手なんですが、どうやって話すんですか？
assistant: まずは、友達に、今話せない状況について、あなたが思っている事を伝えてみて下さい。
user: 話せない状況について何度かLINEで話してるんですけど……  どんな感じで話したら上手くいきますか？
assistant: そうですね…。 相手を責める感じてはなく〈私はこう思っている〉という感じで、話してみてはどうでしょうか。
assistant: [問題提起をする事により、悩みの核心をはっきりさせる為] なぜ、友達は、あなたに色々聞いてもらいたいのでしょうかね?
assistant: 難しいかもしれませんが、相手の気持ちを考えながら、自分の意見を伝えてみてもいいかもしれませんね。
user: 高校生のLINEって、大した話しないんですけど…… だから面倒くさくって…
user: やり取りが多いんですよね
user: 会って話した方が楽だと思いませんか？
user: 楽しいし…
assistant: では、友達にそう伝えてみてはどうですか。
user: でもそんな事言えないですよねーーー
assistant: 言う事で、さらに友達の仲が深まる事もありますよ。
user: 前まではそれで楽しかったんですけど、高校３年生になったし……
assistant: ちなみにお友達も受験を考えられていますか?
user: はい、推薦で行きたいとか言ってます。
user: 高校生になってからの友だちなので気を使う事もあります。
assistant: [何ができるかを考えるため] 一緒に受験勉強してみようと誘って見るとか、1時間だけ話そうなど、時間を区切って、会ってみてはどうですか。
user: LINEじゃなくて直接言うのがいいですかね？
user: でも上手く話す自信がないかもです
assistant: そうですね。 出来ればその方がいいと思いますよ。
user: 最近、話題についていけてないような気もしてるんです
assistant: 言葉にうまくできなくても、表情や声で、伝わる事もありますし、
assistant: 自分自身の事も大切にしつつ、相手との事も大切にしていきたいですよね。
assistant: [何ができるかを考えるため] 話題についていけないのであれば、別の話題をあなたから振ってみるなど、してみてもいいと思いますよ。
user: ずっと高校入ってからマスクだったし、LINEで話してた事の方が多かったし。  どんな話題がいいですかね？
assistant: [何ができるかを考えるため] そうですね‥。 受験生という事ですし、進路の事を話したり、共通の事で話せそうな事があれば、話をしてみればいいのではないでしょうか。
user: 学校では休み時間だけ一緒に過ごしてました。食事もほぼ黙食だったし、なんだか友だちってなんだろう？と先生と話していて思いました。  共通の話題は学校の事くらいかもしれません。
assistant: まずはそれでもいいと思いますよ。
user: とりあえず、気まずい状態をどうやって普通に話せるようにしたらいいですかね？
assistant: 何かきっかけがつかめるといいですよね。
user: はい。
assistant: [何ができるかを考えるため] プリントを渡すとか、飴とかちょっとしたお菓子をあげてみるとか。
user: そう言う事した事なかったです
assistant: [クライアントにこれならできそう、と思ってもらえるため] 些細な事で構わないので、そこから、話を切り出してみてもいいと思いますよ。
assistant: 良かったら、試してみてくださいね。
user: そんな事をしながら、普通に謝ったほうがいいですよね？
assistant: そうですね。相手の様子を見ながら、話してみて下さいね。
user: はい、やってみます
assistant: [勇気づけ] うまくいくといいですね。 応援していますよ。
user: ありがとうございます
assistant: こちらこそ、お役に立った様で良かったです。 また何かあれば、お越しくださいね。
user: はい。ありがとうございます。
assistant: [相談を終了していいか確認する] こちらこそ、相談に来てくれてありがとうございました。 長く話したし、ゆっくり休んで下さいね。では、これで相談は終わりでよいでしょうか。
user: はい。ありがとうございます。
assistant: [Dialogue Finished] ありがとうございました。
